# Gridle - The Ultimate F1 Wordle Game (Formula 1 Driver Guessing Game)

Gridle is a premium, high-performance Formula 1 Wordle clone built using React and Vite. Test your F1 knowledge daily or play in infinite mode to guess the mystery driver based on their team, age, nationality, wins, and career stats.

Designed with a sleek, responsive dark mode, glassmorphic UI elements, dynamic flag/logo icons, a persistent statistics engine, and an interactive hint system, it is the perfect daily puzzle for every motorsport fan.

---

## 🏎️ Key Features

- **Daily F1 Wordle Challenge**: One mystery driver per day, synced globally. Can you guess who it is in 6 tries?
- **Infinite Mode**: Practice as much as you want with a randomized selection of drivers.
- **Interactive Hint System**: Unlock dynamic driver hints as you progress (with intelligent rules based on your game mode).
- **Persistent Statistics Engine**: Tracks played games, win percentage, current streak, max streak, and your exact guess distribution.
- **Responsive & Premium Dark Mode**: High-contrast, glassmorphic UI matching F1's signature red accent aesthetics.
- **Visual Grid Hints**: Instantly see how your guess compares to the target driver (exact matches, directional arrows for age/wins, and flag icons for nationalities).

---

## 🎮 How to Play

1. Type and select a driver from the F1 database.
2. Submit your guess to reveal how close your driver's stats are to the target:
   - **Driver**: Correct name indicator.
   - **Nationality**: Green for correct country.
   - **Team**: Green for correct team.
   - **Age**: Green for correct age, or an arrow indicating if the target driver is older (🔼) or younger (🔽).
   - **Wins**: Green for correct win count, or an arrow indicating if the target driver has more (🔼) or fewer (🔽) career wins.
3. Use hints if you get stuck (unlocked under custom difficulty rules).

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Build the production application:
   ```bash
   npm run build
   ```
